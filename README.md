A URL shortener service that takes a valid URL and returns a shortURL, redirecting the users to the previously provided URL

<h2 align="center">Setup üõ†</h2>  
  
### Prerequisites :  
  
- Make sure you have the LTS version of [Node](https://nodejs.org/) installed.
- Install [git](https://git-scm.com/downloads) for version control.
- Install [MongoDB](https://www.mongodb.com/try/download/community) locally. ([Resource](https://www.mongodb.com/docs/manual/administration/install-community/))

### Installation :

- Step 1: Fork [this repo](https://github.com/jahansid/miniUrl).
- Step 2: Clone your forked version of this repo locally. To clone, go to your command line / terminal, cd over to an appropriate directory and type in `git clone https://github.com/<your username>/miniURL`.
- Step 3: `cd client-app`
- Step 4: While in the `client-app` directory, install the frontend dependencies using `npm install` or `yarn`.
- Step 5: `cd server`
- Step 6: While in the `server` directory, install the backend dependencies using `npm install`.
- Step 7: Add the `.env` file in your `server-app` and add `CONNECTION_URI`.
- Step 8: Go to MongoDB atlas and create your DB and use the connection string.
- Step 9: Go inside the `server-app` directory and run the backend server using `npm start` or using [nodemon](https://www.npmjs.com/package/nodemon). The backend server will start on `localhost:8001`.
- Step 9: While backend server is still running, open a new tab on the terminal and `cd client-app`.
- Step 10: Finally start the react app using `npm start`.

#### Note :

- Don't forget to close all the running servers before closing the terminal/ command line.
- Close all the running servers by visiting to every tab of the terminal the server is running on.
- To close a running server use `ctrl+c`.
